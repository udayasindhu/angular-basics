This application has two components

1. View Users: This component list all the users

2. Add User: This contains a form field to enter the user name

What do you need to do:
  • 1. Using the API "https://jsonplaceholder.typicode.com/users", fetch the users list and show their name, email and phone it in the List Users component
  • 2. Add routing for add-user component
  • 3. In Add User component,
    • Add validation for required and min length for the name form control
    • Display the form value as entered by the user in the html. The form value should be updated simultaeneously.
    • Call a function on button click which will set value "John" in the form control
